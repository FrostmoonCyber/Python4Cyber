# 🛡️ Python4Cyber: Security & Automation Journey

Welcome to my Python learning path focused on Cybersecurity. This repository documents my transition from a Telecommunications Specialist to a Security-focused Developer.

## 🚀 The Project: Network Recon & Audit Tool
The main goal of this repository is to build a modular security tool that automates network reconnaissance tasks.

### 📈 Progression Milestones
- [ ] **Level 1: Input & Validation** - Handle IP addresses and basic user interaction.
- [ ] **Level 2: Connectivity Check** - Automate system pings to verify host availability.
- [ ] **Level 3: Port Scanner** - Implementation of the `socket` library to detect open services.
- [ ] **Level 4: Logging & Reporting** - Exporting scan results to structured files (TXT/CSV).

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Libraries:** `socket`, `os`, `subprocess`, `csv`
- **Focus:** Network Security, Automation, Ethical Hacking

## 📂 Folder Structure
- `/01_Fundamentals`: Basic syntax and initial scripts.
- `/02_Network_Tools`: Scripts for network interaction.
- `/scripts`: Completed, ready-to-use tools.
